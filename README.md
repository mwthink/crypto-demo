# crypto-demo
This repo contains a web application that demonstrates basic
cryptographic functions via PGP including:
- Hashing
- Key generation
- Signing
- Verifying
- Encryption
- Decryption

The application can be [accessed here](https://mwthink.github.io/crypto-demo/).

## Quick Start
To run in dev mode:
```sh
yarn install
npx webpack serve
```
